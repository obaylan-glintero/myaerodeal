Test Welcome Email Function

This tool tests if your welcome email Edge Function is configured correctly.

1. Get Your User ID and Company ID

  1. Go to Supabase Dashboard → Authentication → Users
  2. Copy your User ID
  3. Go to Table Editor → profiles → find your profile
  4. Copy the company_id

2. Get Your Service Role Key

  1. Go to Supabase Dashboard → Settings → API
  2. Copy the service_role key (starts with eyJ...)

3. Test the Function
